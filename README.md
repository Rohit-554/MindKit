## Local AI Runtime

### Android

Android uses Microsoft ONNX Runtime GenAI. Configure the hosted model bundle in
`local.properties`:

```properties
mindkit.model.download.url=https://your-cdn.example/models/model.zip
mindkit.model.checksum.sha256=<sha256>
mindkit.model.expected.size.bytes=<optional-byte-count>
```

The ZIP must contain an ONNX Runtime GenAI model directory, including
`genai_config.json` and its tokenizer/model files. The app downloads, verifies,
extracts, loads, and runs the model from app-private storage.

### iOS

iOS imports Apple's `FoundationModels` framework through the native Swift host.
The model is supplied and managed by Apple Intelligence; the app does not
download it. Availability depends on iOS 26+, eligible hardware, Apple
Intelligence being enabled, and the system model being ready.
